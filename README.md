# angelo-ecommerce-backend
Module 13 Challenge for UT Full-Stack Bootcamp

## Description
This is a set of api routes set up for a E-Commerce web application. It provides a suite of API calls for creating, updating, and deleting Products, as well as their associated Categories and Tags.
